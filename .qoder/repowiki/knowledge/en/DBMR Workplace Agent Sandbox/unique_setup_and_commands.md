Apply the overlay scripts in order, run `alembic upgrade head` and `python -m app.db.seed` (idempotent), then `pytest -q` to validate. Start the server with `python -m uvicorn app.main:app --host 127.0.0.1 --port 8043` and the Angular frontend from `frontend/` via `npm start`. Contract validation uses `python scripts/validate_frontend_contracts.py` and per-phase `validate_angular_phase*.py` helpers.
