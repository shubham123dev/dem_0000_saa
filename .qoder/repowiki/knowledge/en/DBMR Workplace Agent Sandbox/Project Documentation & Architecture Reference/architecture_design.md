This module is a flat collection of Markdown reference documents that serve as the authoritative design contract for the backend. Each file targets a distinct architectural concern: ARCHITECTURE.md defines repository boundaries, exact-schema read/write flows, compatibility bridges, and the NucleusOrganizationGateway port; SECURITY_MODEL.md codifies backend-owned authorization, permission names, secret boundaries, and multi-resource review binding; NUCLEUS_SCHEMA_MAPPING.md documents ORM-to-SQLite table mappings; ORGANIZATION_API_CONTRACTS.md specifies stable HTTP/domain contracts; GOVERNED_ACTION_CONTROL_PLANE.md describes the registry-driven mutation engine; AGENT_* files cover native resources, SSE runs, and tool contracts; TEST_USER1_CUTOVER.md and WORKPLACE_* files document operational procedures. There is no code — these docs are consumed by developers as the single source of truth for contracts between routes, tools, the Nucleus adapter, and the SQLite exact-schema mock.
