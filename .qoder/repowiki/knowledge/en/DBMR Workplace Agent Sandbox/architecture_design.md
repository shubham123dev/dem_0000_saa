The root is an overlay project that patches a base Workplace Agent codebase via idempotent `apply_*` scripts (e.g. `apply_nucleus_gateway_boundary.py`, `apply_angular_phase*.py`, `apply_final_workplace_workflows.py`) against a pinned commit (`1aec2a3bb08f79a8a08782596c59533e42916dfa`). The FastAPI app exposes REST endpoints under `/workplace/organizations/{organization_id}/nucleus/*` plus the standard agent routes, all gated through an authenticated mock-user → active-organization-membership → backend permission → sandbox guard chain. Reads go through the framework-neutral `NucleusOrganizationGateway` protocol backed by an SQLite repository; writes flow through a uniform approval-gated action pipeline (inspect → immutable before/after proposal → approval threshold → permission/fingerprint/precondition revalidation → optimistic version check → idempotent execution → exact reconciliation → audit → optional rollback). Alembic migrations (head `0014_workplace_resources`, then `0015_workplace_workflows`) create exact-schema tables alongside the legacy lowercase tables, with a sidecar `nucleus_resource_versions` table for optimistic concurrency. Tests in `tests/` cover every layer: nucleus gateway, admin control, workplace resources/workflows, agent orchestrator, SSE runs, Angular contract validation, and migration integrity. The Angular frontend lives under `frontend/` and is validated against the backend contracts via `scripts/validate_*` helpers.
