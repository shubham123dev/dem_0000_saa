# Phase 3 workplace shell

Phase 3 replaces the design-system showcase with the production-shaped workplace shell:

- thin brand strip and compact global header;
- searchable grouped primary navigation;
- central workspace with functional section navigation;
- collapsible, resizable desktop Ask AI panel;
- overlay panel at medium widths and full-width panel on compact screens;
- compact navigation drawer with backdrop;
- persisted assistant width and open state;
- keyboard shortcut `Ctrl/Cmd + Shift + A` for Ask AI and Escape for overlays;
- no fake chat, activity, approval, execution, or backend data.

All visible navigation controls perform a real shell action. Live resource projections and conversation behavior remain in their dedicated later phases.
